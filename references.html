HTML

HTML element reference — Mozilla Developer Network
A well-maintained compendium of HTML elements from Mozilla. More of a technical reference and best for when you want to know something specific about a concept you're already a little familiar with.
Dive into HTML5
An introduction to HTML5 and overview of some of its distinguishing features.
HTML Reference at WebPlatform.org
Another extensive reference by Mozilla. Contains a very clear and easy-to-use HTML element reference.
HTML Questions and HTML5 Questions at Stack Overflow
Stack Overflow is a great resource for asking and answering development questions. For a great example, see this question about the <br> tag.
W3School HTML Reference
This is not the best reference on the list but one you will likely come across when turning to Google. Better than it used to be and not bad when you need quick information on a particular element.

CSS

CSS reference — Mozilla Developer Network
A well-maintained CSS property reference from Mozilla.
CSS Reference at WebPlatform.org
Another extensive reference by Mozilla. Contains a very clear and easy-to-use CSS property reference.
CSS Tricks
The Almanac is a particularly great resource. Click on a particular CSS attribute to receive examples of how it's used — text-decoration is a good example.
CSS Questions and CSS3 Questions at Stack Overflow
Stack Overflow is a great resource for asking and answering development questions.
CSS Zen Garden
A beautiful demonstration of how CSS can take the same HTML structure and make it appear in a wide variety of ways.
Lost World Fairs — Atlantis
The entire Lost World Fairs series is fun, but this example in particular shows the creative ways you can leverage CSS to create things.
CSS display Property Explained
These articles — and answer from Stack Overflow — do a good job of explaining and demonstrating how browsers interpret the display property.
  • CSS Tricks — Display
  • What is the difference between display: inline and display: inline-block? — Stack Overflow
  • The Difference Between “Block” and “Inline” — Impressive Webs

JavaScript

JavaScriptEvent Reference
A guide and explanation of various DOM events that take place in the browser.
JavaScript reference — Mozilla Developer Network
A well-maintained JavaScript reference from Mozilla
JavaScript Reference at WebPlatform.org
Another great resource from Mozilla.
JavaScript Questions at Stack Overflow
Stack Overflow is a great resource for asking and answering development questions. For a great example, see this question about the <br> tag.
HTML5 Rocks
A site that focuses primarily on the JavaScript aspect of HTML5. A somewhat advanced but interesting read.

General Reference

Web Platform
Web Platform Docs is a community-driven site that aims to become a comprehensive and authoritative source for web developer documentation.
Can I Use
A good resource for learning which browsers support which features in HTML5 and CSS3.
20 All Too Common Coding Pitfalls For Beginners
A good article highlighting common mistakes and bad habits that can arise when you're beginning to work with JavaScript.
Try Git
GitHub has made an interactive demo to help you learn the basics of git.

Books

O'REILLY Books
Excellent books on a wide variety of programming topics, but I've linked to their HTML/CSS section in particular as it's most relevant to the course. The JavaScript portion would be relevant as well. In particular I'd recommend exploring their cookbook series as they take a project-based approach to teaching principles and practices. Browsing the CSS Cookbook table of contents should give you a good sense of what's in store.
Manning
These books have more a focus on programming than markup (CSS/HTML) and the most relevant books for this course would be on JavaScript. A bit heady and technical at times, but if this aspect of web development interests you I highly recommend it. Secrets of the JavaScript Ninja in particular is quite good. So is jQuery in Action.
A Book Apart
A counterpart to the equally excellent A List Apart blog, these books take in-depth looks at modern web development and design concepts from the perspective of active, successful folks in the indstury. Particularly relevant to this course: CSS3 for Web Designers, HTML5 for Web Designers and Responsive Web Design.

Further Reading

Paul Ford: What is Code?
A well-written article that talks about what "code" is and the author's decades of experience in the industry as a programmer. Smart, funny, well-written. Not a technical reference at all but a very human perspective
A List Apart
A good resource/blog on modern web design practices and principles. Less of a technical reference than other resources on this list but a good all-around resource for anyone who makes things on the web. It's also a cleverly built website in its own right with the source code available on GitHub.

Templates & Frameworks

HTML5 Boilerplate
A popular front-end web template that demonstrates the most common practices to assure compatibility across a wide variety of devices. They also provide a neat template generator that lets you pick and choose the components to start your project with.
Bootstrap
A front-end framework that addresses some of the most common needs for the web. View the examples on the getting started page to get a sense.
